# Power Playground

强大的 Playground，提供一系列可扩展的 API 简化你去构建一个属于自己的 Playground 的成本。

* 配置方式简单多样，适配你的各个角度的定制需求
* 支持多种语言，JavaScript, TypeScript, TSX, ...
* 插件丰富可扩展性强，通过丰富的 API 可以实现各种各样的功能
* 底层技术栈强大，基于 Monaco Editor、Chrome DevTools 实现，支持各种各样的编辑运行需求
