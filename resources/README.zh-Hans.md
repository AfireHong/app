# Power Playground

|zh-Hans
|[en-US](../README.md)

提供扩展性强大的 API 简化构建一个属于你项目的优秀 Playground 的成本。

* 配置方式简单多样细致，配合你各个角度的定制需求
* 支持多种语言，JavaScript, TypeScript, TSX, ...
* 插件丰富可扩展性强，通过丰富的 API 可以实现各种各样的功能
* 底层技术栈强大，基于 Monaco Editor、Chrome DevTools 实现，支持各种各样的编辑与运行的需求
